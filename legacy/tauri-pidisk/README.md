# Legacy Tauri PIdisk

This directory contains the original PIdisk implementation built on Tauri 2 + Rust + React.

**Status:** frozen for reference, not built.

The project has been rewritten on Go 1.26 + Wails v2 + React + MUI. The new sources live in the repository root. Files here exist purely as a behavioural reference: when the new implementation needs to match the original UI flow or a specific SFTP edge case, consult the relevant file in this directory.

Do not modify these files. Do not include this directory in the build.
